# Overview

This repository teaches agent systems engineering by combining structured study with real system building.

The core philosophy is simple: learn by building systems that are useful, testable, and maintainable. Concepts are introduced only to support implementation quality, evaluation discipline, and operational reliability.

The progression is deliberate:

1. Single agent systems
2. Modular systems
3. Multi-agent systems
4. Decision systems

Each stage adds capability without losing clarity. The goal is not to maximize complexity, but to engineer systems that behave well under real constraints.
