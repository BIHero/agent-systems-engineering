# Agent Systems Engineering

Agent Systems Engineering is a structured curriculum and working portfolio for mastering agentic AI by building real systems.

The repository is designed around production readiness. Each layer and project emphasizes evaluation, reliability, observability, and practical system design rather than demos alone.
